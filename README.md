# Chomsky Hierarchy Visualization

An interactive web-based visualization of the **Chomsky Hierarchy**, explaining formal languages and their computational models.

---

## Features

* Interactive hierarchy diagram
* Expandable sections for each language type
* Automata visualizations (DFA, PDA, LBA, Turing Machine)
* Grammar rules, examples, and theory

---

## Tech Stack

* HTML5
* CSS3
* JavaScript
* SVG

---

## How to Run

```bash
git clone https://github.com/your-username/chomsky-hierarchy.git
cd chomsky-hierarchy
```

Open `index.html` in your browser.

---

## Concept

Based on the work of Noam Chomsky , classifying languages into:

* Type 0 (RE)
* Type 1 (CS)
* Type 2 (CF)
* Type 3 (Regular)
